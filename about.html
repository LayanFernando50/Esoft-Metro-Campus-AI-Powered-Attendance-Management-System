📘 About the System

Esoft Metro Campus AI-Powered Attendance Management System

This AI-Based Facial Recognition Attendance System is designed exclusively for Esoft Metro Campus – Negombo.

❓ Why Was This System Created?

  • Eliminates the need for ID cards or manual logs
  • Prevents proxy attendance
  • Sends real-time notifications
  • Enhances campus security and reputation
  • Protects student data with encryption

💡 What the System Does

  • Scans student faces in real-time
  • Verifies identity with deep learning
  • Automatically marks attendance
  • Sends SMS notifications
  • Logs and secures attendance data
  • Flags students without scheduled lectures

🌟 Core Features

🎯 AI Recognition

Matches faces under varying conditions using AI.

🕒 Real-Time Logs

Instant attendance with course timestamps.

📲 SMS Alerts

Sends live alerts for entry/exit.

🗓️ Schedule Awareness

Detects unscheduled visits.

📊 Admin Reports

Generate and export attendance data.

🔐 Data Privacy

Encrypted storage and role-based access.

👥 Who Uses the System?

User RoleFunction
🧑‍🎓 StudentsWalk in and get marked present automatically
👩‍🏫 LecturersMonitor logs and schedule matching
👨‍💼 AdminsManage profiles, reports, configurations
👮 SecurityMonitor students without lectures

⚙️ How It Works

  1. Student enters the campus
  2. Camera scans face
  3. AI matches it to the database
  4. Attendance is marked
  5. Schedule is validated
  6. Notification is triggered
  7. Admin can review logs

🧪 Technology Stack

  • Python, OpenCV, TensorFlow
  • MySQL or Firebase
  • Flask / Django
  • Twilio / FCM
  • React / HTML / CSS

🔐 Data Privacy and Security

Facial and attendance records are encrypted. Access is restricted and complies with modern data protection standards.
